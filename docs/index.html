Worms
The game was developed as a project coursework. For a month and a half, 2 friends and I had to implement the whole game from scratch, which included the client, server, communication protocol, game logic and physics simulation.
We developed the whole project in C++, using SDL2pp for the user interface, TCP/IP sockets for the networking protocol, and Box2D to simulate the physics.
Features
The game has several features, including
  • Create multiple matches
  • Multiplayer matches (up to 4 players)
  • Several weapons (bazooka, grenade, dynamite and... BANANA!)
  • Choose between different maps
  • Cheats (infinite hp, infinite ammo, etc)
